# Dokkan Battle Unit Guide — PRD (Product Requirements Document)

## Product Goal
A public-facing iPhone app that helps Dokkan Battle (Global) players quickly find units, understand which team slot they fit best, and view their key stats & skills.

## Target Users
Anyone who plays Dokkan Battle Global and wants a quick in-game reference guide on their phone.

## My Experience Level
Beginner — some HTML/CSS basics. Explain steps clearly, avoid jargon, and guide me through one step at a time.

## Core Features (MVP)
1. **Unit Search** — search bar to find units by name instantly
2. **Slot Guide** — each unit shows which team slot they fit best (Leader, Sub, Friend)
3. **Unit Stats & Skills** — display type, rarity, passive skill, active skill, and links

## Out of Scope for MVP
- Unit images / artwork (text only for now)
- Japan version units
- Team builder
- Tier lists
- User accounts or saving favourites
- Android version

## Design
- Theme: Dragon Ball inspired — dark background, gold accents
- Platform: iOS only (Expo Go for testing)
- Text only, no images, keep it fast and simple

## Data Source
- Dokkan Battle Global community API (free, no account needed)

## Screens
1. **Home / Search Screen** — search bar + scrollable list of units
2. **Unit Detail Screen** — slot recommendation + stats + skills

## Timeline
- Project setup, connect to API, display unit list, search working
-  Slot guide logic, stats & skills screen, dark/gold styling

## How to Help Me
- I am using React Native with Expo
- Guide me one step at a time
- Show me the full code for each file when making changes
- Explain what each piece of code does in simple terms
- If something can break, warn me in advance
